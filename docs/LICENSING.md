# Licensing Decision

VIO is licensed under GPL-3.0-only for public distribution. Separate commercial
licenses are available from the project owner.

Before public release:

1. Keep the canonical GPLv3 `LICENSE` file in the repository.
2. Add `GPL-3.0-only` SPDX metadata to package recipes where supported.
3. Review third-party dependency licenses and notices.
4. Decide whether a CLA, DCO, or another contribution mechanism is required.
5. Update VXrepo package recipes with the selected license and commercial
   licensing notice.

Legal review is recommended before accepting external contributions or
publishing binaries.
